# TestSprite tests – Nirvana Valentine Webinar

- **Test plan:** [test-plan.md](./test-plan.md) – step-by-step cases derived from `testsprite-prd.md`.
- **Setup and run:** See project root [TESTSPRITE.md](../TESTSPRITE.md).

## Quick run

1. Add and connect **TestSprite MCP** in Cursor (Settings → MCP; use your [TestSprite API key](https://www.testsprite.com/dashboard)).
2. Start the app: `npm run dev` (app at http://localhost:3000).
3. In Cursor chat: *"Can you test this project with TestSprite?"* or *"Help me test this project with TestSprite."*
4. In the TestSprite configuration page use:
   - **Application URL:** `http://localhost:3000`
   - **PRD:** upload or paste `testsprite-prd.md`

Reports/artifacts will appear here or as TestSprite reports.
